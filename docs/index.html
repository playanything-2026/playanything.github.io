Abstract

Vision-language model (VLM) driven game agents have shown strong performance in specific games, yet existing frameworks require extensive manual crafting of game-specific prompts and action functions. This reliance results in near-zero cross-game generalization. Therefore, we propose PlayAnything, a plug-and-play multimodal prompt adaptation framework that enables VLM agents to generalize to diverse, unseen games. By combining automatic prompt generation with guided non-expert refinement, PlayAnything reduces deployment effort from hours of expert programming to a streamlined human-AI collaborative pipeline.

Specifically, PlayAnything consists of three complementary modules: i) Gather, a multimodal retriever that constructs structured game profiles from gameplay videos, screenshots, and strategy texts; ii) Adapter, a core module fine-tuned on Qwen3-VL-8B that transforms these profiles into executable prompts and action functions; and iii) Guider, a post-generation verification module that cross-references outputs with raw profiles to provide targeted improvement suggestions for human users.

Experiments demonstrate that PlayAnything, trained on only 5 games, achieves 100% runnability and an average success rate of 58% across 40 unseen games. This drastically outperforms existing baselines—where Cradle and CombatVLA score 0%, and the massive NitroGen reaches only 42%—achieving superior generalization with orders-of-magnitude less training data (5 games vs. 40,000 hours). Furthermore, with Guider-directed refinement requiring a median of only 136 character edits, the average success rate further rises to 67.5%, peaking at 92% on Level 2 tasks.

Framework Overview

Overview of the PlayAnything framework. Gather retrieves multimodal game information and constructs a structured game profile. Adapter transforms the profile into all prompts and action functions required by the agent framework. Guider verifies outputs against the original game profile and produces targeted improvement suggestions.

🔍

Gather

Retrieves gameplay videos, screenshots, and strategy texts from external sources and constructs structured game profiles via a fully automated pipeline.

⚙

Adapter

Fine-tuned on Qwen3-VL-8B-Instruct with only 5 training games. Transforms multimodal game profiles into structured JSON prompts and Python action functions.

🧭

Guider

Performs cross-reference coverage analysis between Adapter outputs and raw game profiles. Outputs structured improvement suggestions for efficient human refinement.

Key Findings:

  1. PlayAnything overcomes the zero-generalization barrier. Both Cradle and CombatVLA achieve 0% Runnability, while Ours achieves 100% across all 40 test games.
  2. Significant advantage with minimal training data. Trained on only 5 games, Ours improves the Avg. SR from 42.0% (NitroGen, 40K hours) to 58.0%.
  3. Guider-directed refinement is efficient and effective. With a median of only 136 character edits, Ours (Refined) pushes SR2 to 92.0% and Avg. SR to 67.5%.

Conclusion

  1. Multimodal adaptation enables effective generalization with minimal data. Learning prompt generation from just 5 games allows PlayAnything to generalize across 40 unseen titles. This proves structured adaptation is a highly efficient alternative to massive pre-training, eliminating the need for per-game manual engineering.
  2. A new large-scale benchmark validates cross-type robustness. We contribute a 45-game benchmark across four types—4x the scale of prior state-of-the-art evaluations. Results demonstrate PlayAnything's robustness even in challenging scenarios like coordinate-based control, where fixed action-space models fail.
  3. Collaborative refinement empowers non-experts. Our human-AI pipeline achieves a 67.5% success rate with a median of just 136 character edits. This drastically lowers the barrier to deploying and optimizing sophisticated VLM agents for any new game.
